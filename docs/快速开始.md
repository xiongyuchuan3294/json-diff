# 快速开始（长期运行）

仅保留最短命令清单，适合内网长期跑回归。

## 1) 首次一次性初始化（建回归支撑表）

```bash
cd /path/to/json-diff
export MYSQL_DEMO_CONF='<host>,<port>,<database>,<user>,<password>,utf8mb4'
python3 scripts/init_schema_only.py
```

## 2) 每台机器一次性准备 Python 依赖

```bash
cd /path/to/json-diff
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
```

## 3) 每次执行回归（固定 3 参数）

```bash
cd /path/to/json-diff
source .venv/bin/activate
export MYSQL_DEMO_CONF='<host>,<port>,<database>,<user>,<password>,utf8mb4'
export REGRESSION_TARGET_SCHEMA='rrs_test_dev'
python3 scripts/regression_cli.py "ALL" "<old_scenario_id>" "<new_scenario_id>"
```

## 4) 查看结果

```bash
cat output/latest.md
```
